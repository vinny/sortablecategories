# Sortable Categories [![Tests](https://github.com/vinny/sortablecategories/actions/workflows/tests.yml/badge.svg)](https://github.com/vinny/sortablecategories/actions/workflows/tests.yml)

A phpBB extension that allows board users to reorganize forum categories using drag and drop on the index page

## Installation

1. Copy the contents of this repository to `ext/vinny/sortablecategories/` inside your phpBB installation.
2. Go to the ACP -> **Customise** -> **Manage extensions**.
3. Locate **Sortable Categories** under the disabled extensions list and click **Enable**.
